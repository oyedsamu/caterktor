# Module caterktor-serialization-cbor

CBOR content negotiation for CaterKtor using kotlinx.serialization.

This module supports compact binary payloads while keeping the same request, result, and converter contracts used by the other serialization modules.

# Package io.github.oyedsamu.caterktor.serialization.cbor

CBOR converter and DSL entry points. Install these APIs to register CBOR body conversion with CaterKtor's content negotiation registry.
