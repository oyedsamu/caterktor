# Module caterktor-serialization-json

JSON content negotiation for CaterKtor using kotlinx.serialization.

This module provides JSON body conversion, request enveloping helpers, and response unwrappers for common API shapes such as `data` envelopes.

# Package io.github.oyedsamu.caterktor.serialization.json

JSON converter and DSL entry points. Install these APIs when your service sends or receives `application/json` payloads and you want typed CaterKtor helpers to encode request bodies and decode response bodies automatically.
