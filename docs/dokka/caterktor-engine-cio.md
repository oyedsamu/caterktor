# Module caterktor-engine-cio

CIO-backed Ktor engine integration for CaterKtor.

Use this module for multiplatform targets where Ktor's CIO engine is the desired runtime transport.

# Package io.github.oyedsamu.caterktor.engine.cio

CIO engine factory helpers for CaterKtor. These APIs keep Ktor engine selection explicit and separate from the core request pipeline.
