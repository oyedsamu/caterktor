# Module caterktor-testing

Test utilities for CaterKtor clients and integrations.

This module provides fakes, assertions, and server-style helpers for validating request construction, response handling, and interceptor behavior without making real network calls.

# Package io.github.oyedsamu.caterktor.testing

Fake transports, fake clients, assertions, and test server utilities. Use these APIs in unit tests for repositories, SDK clients, and custom interceptors built on top of CaterKtor.
