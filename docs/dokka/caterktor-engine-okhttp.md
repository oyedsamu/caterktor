# Module caterktor-engine-okhttp

OkHttp-backed Ktor engine integration for CaterKtor.

Use this module when a JVM or Android application wants Ktor's OkHttp engine behind the CaterKtor pipeline.

# Package io.github.oyedsamu.caterktor.engine.okhttp

OkHttp engine factory helpers for CaterKtor. These APIs keep engine selection outside of `caterktor-core` while giving applications a small, discoverable way to install the OkHttp transport path.
